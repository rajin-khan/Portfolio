# Tech icon sources

These SVGs are stored locally so the tech-stack marquee has no runtime image
dependencies. They were optimized with SVGO 4.0.0 after download.

- Devicon v2.17.0: Git, Vite, Astro, React, Tailwind CSS, AWS, MySQL,
  Flutter, FastAPI, NumPy, Pandas, Matplotlib, PyTorch, OpenCV, Jupyter,
  C, C++, Java, Python, Dart, PHP, HTML, and CSS. Astro, AWS, and Jupyter
  include dark-background variants derived from the same source artwork.
- Simple Icons v16.21.0: Railway, Vercel, Next.js, Stripe, Lemon Squeezy,
  Paddle, Mintlify, Supabase, WordPress, Ollama, and Bash.
- Codex: the official dark Codex app icon bundled with the OpenAI desktop
  app, resized to 128px for the marquee.
- Cursor: the official 2D dark-background cube mark from Cursor's brand kit.
- Groq: the circular Groq avatar previously published on Hugging Face.
- Pandas dark variant: the official white Pandas mark from
  `https://pandas.pydata.org/static/img/pandas_mark_white.svg`.
- Chroma: the favicon from `chroma-core/chroma` commit
  `3dea03ca4bde0a7ace6d651a6ea80e0e4212b413`.
- Seaborn: the dark-background mark from `mwaskom/seaborn` commit
  `f04b6cd5484267a0885d1fed068e99dff3a1b226`.
- Ultralytics YOLO: the official Ultralytics YOLO mark.
- SQL: Lucide's database icon.
